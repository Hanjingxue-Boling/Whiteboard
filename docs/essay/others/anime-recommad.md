# 动画推荐

以下是一些动画推荐，推荐给想要了解动画这一种文化或者说艺术形式的人。

我喜欢看动画，也就是网民常常说的动漫。它确实存在幼稚的特征（动画自始至终的主要受众是青少年），但我也喜欢它所具有的其他艺术形式所无法表达的内容，技法又或者是其他瑰丽的东西。
